Basic

here's a step-by-step guide on how to create an application on Jamilx:

Method 1: Using Command-Line Tool

Open the command prompt or terminal on your computer. Navigate to your Jamilx folder by using the cd command. Run the following command: jamilx CreateApp "App Nickname" "App Name" "App Description" Hit Enter and wait for the command to finish executing. The command will create a boilerplate for your application in the Apps directory. Your application's data will be stored in this directory.

Method 2: Self-Creation

Go to the Apps directory. Create a new directory with your app nickname. Inside the directory, create a PHP file with the app nickname as the file name. In the PHP file, create a class that extends JXService and implements JXServiceI. Create a conf.json file and fill in the necessary fields for your app. You can use the demo app as a sample.

Method 3: GUI

Go to your Jamilx admin interface. Navigate to the "app creation button" and click it. Follow the on-screen instructions to create your app. Fill in the necessary fields, such as app nickname, app name, and app description. After creating your app using any of these methods, you can start developing your application. You can add new services, actions, and prototypes to your app, as well as customize its user interface using the Container layer. You can also use the command-line tools provided by Jamilx to automate repetitive tasks and streamline the development process.

Jamilx provides multiple ways to create an application, allowing developers to choose the method that works best for them. The framework is flexible and powerful, making it easy to develop and maintain web applications in a fast and efficient manner.
