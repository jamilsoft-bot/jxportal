PCS Pattern

Prototype-Container-Service (PCS) pattern

Jamilx is a PHP framework for building rapid application development (RAD) and software-as-a-service (SaaS) applications. It follows a Model-View-Controller (MVC) architecture using the Prototype-Container-Service (PCS) pattern. The framework includes three useful interfaces that all other classes must implement to ensure consistency: JXServiceI, JXActionI, and JXPrototypeI.
